## Development install

install dependencies and run bower

		npm install

start and watch

		npm start

Test server at [http://space-boats.heroku.com](http://space-boats.heroku.com)

## Editing Maps
1. Edit data-src/*.tmx files
1. Export *.json in src/data/*.json
1. run "npm start" or "npm run build".
